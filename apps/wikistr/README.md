# wikistr

Wikistr has moved to the [Nostroots](https://github.com/Trustroots/nostroots) repo.

Use the app at [https://nos.trustroots.org/examples/wikistr/](https://nos.trustroots.org/examples/wikistr/).

This directory only publishes a redirect for the legacy GitHub Pages host at
`https://wikistr.trustroots.org/`. Hash routes such as `#hitchwiki` are
preserved when redirecting.
